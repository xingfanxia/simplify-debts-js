Split Debt Better!

A simple debt simplifier that can reduce any number of transactions among n people to n-1 transactions.

How to use

Enter your graph definition in the textarea below. Use the following format:

# Everyone owes Joe (Including Joe himself, i.e. Joe paid for everyone involved for a bill)
* -> Joe: 16.80

# Sue owes Joe
Sue -> Joe: 24.40

# Bob owes Sue
Bob -> Sue: 12.20

# Ellen is part of all the debt and didn't pay for anything
Ellen
